# branch-app

react-native app which connect to `branch-be` node app

## Dev environment


### stack

- react-native
- react-navigate v6


### Runing project

``yarn install``

## run ios

Install cocoapods

``sudo gem install cocoapods``
``brew install watchman``

``remove ios/Podfile.lock``

``cd ios``

``pod install``

In a terminal run 

``npx react-native start``

then run

``npx react-native run-ios``
