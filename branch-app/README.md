# branch-app

react-native app which connects to `branch-be` node app

## Dev environment


### stack

- react-native
- react-navigate v6


### Running project

``yarn install``

## run ios

Install cocoapods

``sudo gem install cocoapods``
``brew install watchman``

``remove ios/Podfile.lock``

``cd ios``

``pod install``

In a terminal run 

``npx react-native start``

then run (IOS)

``npx react-native run-ios``

and/or run (Android)

``npx react-native run-android``
